# ggen
ggen is password generator written by Pure Go language.

# Usage

```shell
ggen
```

`-f` you can customize password characters

| Specifier | Description |
|---|---|
| `L` | Lower letters |
| `U` | Upper letters |
| `D` | Digits ( 0~9 ) |
| `S` | Symbols ``~!@#$%^&*()_+`-={}|[]\\:\"<>?,./`` |

`-n` number of length. default length is 8.

`-uuid` generated UUID v4
